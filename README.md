##  Heart Disease Prediction Using Machine Learning

## Project Overview
Heart disease is one of the leading causes of death worldwide. Early prediction and diagnosis can significantly reduce the risk and improve patient outcomes.  
This project uses **Machine Learning techniques** to predict the likelihood of heart disease based on patient health parameters.

The system applies data preprocessing, feature encoding, and classification algorithms to analyze medical data and provide accurate predictions.


## Objectives
- To analyze medical data related to heart disease
- To build a predictive model using Machine Learning
- To improve early detection of cardiovascular disease
- To assist healthcare professionals in decision-making

## Machine Learning Techniques Used
- Label Encoding  
- Random Forest Classifier  
- K-Fold Cross Validation  


## Dataset
- Medical dataset containing patient health attributes
- Features include age, sex, blood pressure, cholesterol levels, etc.
- Dataset is preprocessed to handle categorical values and missing data


##  Technology Stack
- Python  
- Data Science & Machine Learning  
- Pandas, NumPy  
- Scikit-learn  
- Data Analytics  

## System Workflow
1. Data Collection  
2. Data Preprocessing  
3. Feature Encoding  
4. Model Training  
5. Model Evaluation  
6. Heart Disease Prediction  

## Results
- The Random Forest model achieved promising accuracy
- K-Fold validation ensured model reliability
- The system effectively predicts the risk of heart disease

- Can support doctors and healthcare providers
